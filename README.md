# GreenTech Solutions Final Project

## Project Description

GreenTech Solutions is a responsive and accessible website designed for a fictional environmental sustainability company. The site promotes eco-friendly technology solutions that help businesses reduce their carbon footprint and improve sustainability practices.

The website was developed as a final project for ICT 280 and demonstrates front-end web development skills using HTML5, CSS3, JavaScript, responsive design principles, and accessibility best practices.

---

## Features

- Responsive mobile-friendly design
- Accessible navigation with skip links
- Product search and filtering using JavaScript
- Contact form validation
- Sustainability resources and educational content
- Company timeline and mission information

## Setup Instructions

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in a web browser or use Live Server.
4. Navigate through the website using the navigation menu.
